## 📝 简介 | Introduction

**中文：** 文件智能整理工具是一款跨平台的桌面应用，可以帮助你自动扫描指定文件夹中的文件，并根据文件类型智能分类到对应的子文件夹中。支持 macOS 和 Windows 系统。

**English:** File Organizer Tool is a cross-platform desktop application that automatically scans folders and intelligently sorts files into categorized subfolders by type. Supports macOS and Windows.

## ✨ 功能特点 | Features

**中文：**
- **智能分类**：自动识别文档、图片、视频、音频、压缩包、代码等 10+ 种文件类型
- **跨平台**：支持 macOS（原生 .app）和 Windows 系统
- **图形界面**：简洁直观的 Tkinter 图形界面，操作简单
- **快速高效**：批量处理大量文件，毫秒级完成分类
- **完全开源**：代码完全开放，安全透明

**English:**
- **Smart Categorization**: Auto-detects 10+ file types — documents, images, videos, audio, archives, code, and more
- **Cross-Platform**: Works on macOS (native .app) and Windows
- **GUI**: Clean and intuitive Tkinter interface
- **Fast & Efficient**: Batch-process large numbers of files in milliseconds
- **Open Source**: Fully transparent, MIT licensed

## 🔧 使用方法 | Usage

### macOS（中文）
1. 下载 `文件智能整理工具_v1.0_macOS.zip` 并解压
2. 将 `文件智能整理工具.app` 拖入「应用程序」文件夹
3. 首次打开可能提示「无法验证开发者」，请前往「系统设置 → 隐私与安全性 → 仍要打开」
4. 启动后选择要整理的文件夹，点击「开始整理」即可

### macOS (English)
1. Download and extract `文件智能整理工具_v1.0_macOS.zip`
2. Drag `文件智能整理工具.app` into the `Applications` folder
3. If you see "unidentified developer" warning, go to **System Settings → Privacy & Security → Open Anyway**
4. Launch the app, select a folder, and click "开始整理" (Start)

### Windows（中文）
1. 下载 `文件智能整理工具_Windows版.zip` 并解压
2. 双击运行 `一键运行.bat`
3. 或在命令行执行 `python file_organizer.py`

### Windows (English)
1. Download and extract `文件智能整理工具_Windows版.zip`
2. Double-click `一键运行.bat`
3. Or run `python file_organizer.py` in terminal
