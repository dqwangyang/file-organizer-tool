# 文件智能整理工具

> 一键整理桌面文件，智能分类归档

---

## 📝 简介

文件智能整理工具是一款跨平台的桌面应用，可以帮助你自动扫描指定文件夹中的文件，并根据文件类型智能分类到对应的子文件夹中。支持 macOS 和 Windows 系统。

## ✨ 功能特点

- **智能分类**：自动识别文档、图片、视频、音频、压缩包、代码等 10+ 种文件类型
- **跨平台**：支持 macOS（原生 .app）和 Windows 系统
- **图形界面**：简洁直观的 Tkinter 图形界面，操作简单
- **快速高效**：批量处理大量文件，毫秒级完成分类
- **完全开源**：代码完全开放，安全透明

## 🔧 使用方法

### macOS
1. 下载 `文件智能整理工具_v1.0_macOS.zip` 并解压
2. 将 `文件智能整理工具.app` 拖入「应用程序」文件夹
3. 首次打开可能提示「无法验证开发者」，请前往「系统设置 → 隐私与安全性 → 仍要打开」
4. 启动后选择要整理的文件夹，点击「开始整理」即可

### Windows
1. 下载 `文件智能整理工具_Windows版.zip` 并解压
2. 双击运行 `一键运行.bat`
3. 或在命令行执行 `python file_organizer.py`

## 🧩 文件分类规则

| 类别 | 文件格式 |
|------|----------|
| 文档 | .doc, .docx, .pdf, .txt, .md, .xls, .xlsx, .ppt, .pptx |
| 图片 | .jpg, .jpeg, .png, .gif, .bmp, .webp, .svg, .ico |
| 视频 | .mp4, .avi, .mkv, .mov, .wmv, .flv |
| 音频 | .mp3, .wav, .flac, .aac, .ogg |
| 压缩包 | .zip, .rar, .7z, .tar, .gz |
| 代码 | .py, .js, .html, .css, .java, .cpp, .c, .go, .rs |
| 其他 | 未匹配类型的文件 |
